Section 4 - Understanding Pointers

What are Pointers?
Why does this feature exist?
How I can work with Pointers?


Variables that are store values addresses instead of values

You can avoid unnecessary value copies
Directly mutate values

& = makes the variables a memory address instead of that value

* = changes the memory pointer back into the actual value

EXAMPLE BELOW: 
func main() {
	age := 32

	agePointer := &age

	fmt.Println("Age: ", agePointer)
	fmt.Println("Age: ", *agePointer)

	fmt.Println(getAdultYears(agePointer))
}

func getAdultYears(age *int) int {
	return *age - 18
}
